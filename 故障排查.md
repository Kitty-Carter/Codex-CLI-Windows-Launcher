# 故障排查

## PowerShell 出现 `>>`

说明 PowerShell 进入了多行输入状态。

解决：按 `Ctrl + C`，直到回到 `PS ...>` 开头的正常提示符。

## Codex CLI 能用，Cursor Codex 不行

先关闭 Cursor，然后双击：

```text
C:\DIY\Codex CLI\启动Cursor-Codex.cmd
```

## Cursor Codex 提示 sandbox 问题

优先使用 backup sandbox。

如果一直失败，检查：

```text
C:\Users\zyp31\.codex\config.toml
```

建议 Windows sandbox 使用：

```toml
[windows]
sandbox = "unelevated"
```

## Codex 命令写错

错误：

```powershell
codex -p "问题"
```

正确：

```powershell
codex exec "问题"
```
